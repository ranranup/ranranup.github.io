javascript 数据转换

  1. Boolean()
    对任何数据类型的值调用Boolean()函数，都会返回一个Boolean值，返回值是true和false取决于要转换值的数据类型和其实际值。
    • Boolean
      • false 转化为 false
    • String
      • “” 转化为 false
    • Number
      • 0 和 NaN 转化为 false
    • Object
      • null 转化为 false
    • Undefined
      • undefined 转化为 false
      • n/a 转化为 true（n/a 或 N/A, not application ,不适用）
